# Library Management System

## Описание

Этот проект представляет собой систему управления библиотекой с использованием базы данных PostgreSQL. Проект включает в себя Python-скрипты для выполнения операций CRUD (создание, чтение, обновление, удаление) с базой данных книг.

## Структура проекта

- db_connection.py: Содержит класс DataBase для управления подключением к базе данных и выполнения SQL-запросов.
- db_table.py: Содержит абстрактный базовый класс DbTable, который определяет общие операции для работы с таблицами базы данных.
- books.py: Содержит класс Books, представляющий таблицу books в базе данных и методы для выполнения операций CRUD.
- main.py: Основной скрипт для взаимодействия с пользователем и выполнения операций с базой данных.
- docker-compose.yml: Конфигурационный файл для Docker Compose, который настраивает и запускает контейнер PostgreSQL.

## Требования

- Docker
- Docker Compose
- Python 3.x
- PostgreSQL

## Установка

1. Клонируйте репозиторий:


    git clone https://github.com/your-repo/library-management-system.git
    cd library-management-system


2. Запустите контейнер PostgreSQL с помощью Docker Compose:


    docker-compose up -d


3. Установите необходимые Python-библиотеки:


    pip install psycopg2-binary


## Использование

1. Запустите основной скрипт:


    python main.py


2. Следуйте инструкциям в командной строке для выполнения различных операций, таких как отображение всех книг, поиск книги, добавление новой книги, удаление книги и изменение статуса книги.
